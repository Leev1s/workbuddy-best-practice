# Project Memory

## Important boundary

这是人工维护的项目级经验记录，用于帮助后续维护者理解仓库演进。它不是 WorkBuddy 官方 Memory 功能，不代表 WorkBuddy 会自动读取或使用本文件。

## Stable decisions

- 本仓库是非官方中文 WorkBuddy best-practice 知识库，不是官方文档镜像。
- 以“从一句话办公任务，到可复用的企业 Agent 工作流”为核心定位。
- 核心方法论为：Idea → Task Brief → Context Package → Manual Run → Result Review → Workflow Template → Expert / Skill / Connector → Automation → Cloud Agent。
- `.workbuddy/` 作为本仓库实践约定的项目级上下文目录，只提交稳定、可公开、可复用材料。
- 多媒体资产放在 `assets/`，README 使用 WebP 版本，保留 PNG 原图。

## Risks to keep watching

- 不要把 Claude Code 的 `.claude/*` 机制映射为 WorkBuddy 官方机制。
- 不要把 `.workbuddy/` 写成官方规范或自动读取目录。
- 不要将第三方 Skill、Connector、MCP server、安全性、定价或路线图写成官方事实。
- 不要提交私密任务上下文、临时日志、缓存、API Key 或客户数据。
