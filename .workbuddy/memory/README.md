# Project Memory Folder

本目录只用于保存稳定、可公开、可审查的项目级约定。它不是 WorkBuddy 官方 Memory 功能，也不代表 WorkBuddy 会自动读取这里的内容。

可以记录：

- 长期写作规范。
- 术语统一规则。
- 已确认的来源边界。
- 可复用的项目维护经验。

不得记录：

- 私人信息、账号、token、cookie、内部系统凭据。
- 客户资料、未脱敏员工信息、商业机密。
- 临时任务日志、运行缓存、一次性会话内容。
- 任何不适合提交到公开仓库的信息。
