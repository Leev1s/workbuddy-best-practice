# Templates

本目录收集 WorkBuddy 最佳实践中可直接复制使用的短模板。模板用于启动任务、验收结果、评估能力和降低权限风险，不替代企业内部审批、合规审计或官方产品文档。

## Template Index

| Template | Use it for | Risk level | Related guide |
|---|---|---|---|
| [Task Brief Template](task-brief-template.md) | 将办公需求整理成可执行、可验收的 WorkBuddy 任务。 | Medium | [Getting Started](../01-getting-started/) |
| [Result Review Checklist](review-checklist.md) | 在接受、分享或归档输出前进行质量与安全检查。 | Medium | [结果验收指南](../01-getting-started/result-review.md) |
| [Expert Card Template](expert-card-template.md) | 定义 Expert / Expert Team 的职责、输入、输出和边界。 | Medium | [Experts](../04-experts/) |
| [Skill Evaluation Template](skill-evaluation-template.md) | 评估 Skill 来源、权限、数据流和测试结果。 | High | [Skills](../05-skills/) |
| [Connector Evaluation Template](connector-evaluation-template.md) | 评估 Connector 权限、账号、数据流和撤销方式。 | High | [Connectors](../06-connectors/) |
| [Automation Design Template](automation-template.md) | 将已验证工作流整理成可定时、可回滚的 Automation 设计。 | High | [Automation](../07-automation/) |
| [Security Checklist](security-checklist.md) | 在涉及权限、文件、外部服务或无人值守流程前做最低安全确认。 | High | [Security](../08-permission-security/) |
| [WorkBuddy Context Templates](workbuddy-context/) | 组织公开 Context Package、维护 brief、review checklist 和 prompt。 | Medium | [Context Convention](../docs/workbuddy-context-convention.md) |
