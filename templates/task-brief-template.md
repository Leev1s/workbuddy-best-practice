# WorkBuddy Task Brief Template / 任务简报模板

在启动 WorkBuddy 任务前复制本模板，用来把“想让它帮忙”改写成可验收的任务说明。适合文件整理、报告生成、信息抽取、流程设计等办公任务。高风险任务应同时使用安全检查清单。

## Minimal Task Brief

```markdown
目标：
输入：
输出：
工作目录：
允许操作：
禁止操作：
验收标准：
风险提醒：
```

## Definition of Done

- [ ] Output exists in the requested location.
- [ ] Output matches the requested format.
- [ ] All required inputs were used.
- [ ] Assumptions and uncertainties are listed.
- [ ] High-risk operations were confirmed by a human.
- [ ] Result is ready for the target audience.

## Review Checklist

- [ ] Goal satisfied.
- [ ] Input coverage checked.
- [ ] Output quality checked.
- [ ] File paths checked.
- [ ] Changes checked.
- [ ] External calls / data sharing checked.
- [ ] Human approval completed if needed.
